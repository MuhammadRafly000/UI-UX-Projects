# UI/UX Project
Dokumentasi ini berisi rangkuman hasil pengerjaan berbagai proyek UI/UX yang dikembangkan dalam mata kuliah Web Development. Setiap proyek dirancang untuk melatih kemampuan dalam membuat antarmuka yang menarik, memecahkan masalah pengguna, serta menerapkan prinsip-prinsip UX dalam berbagai konteks website maupun aplikasi.

Proyek-proyek yang dikerjakan mencakup beragam kategori, mulai dari website layanan, landing page bisnis, platform informasi, e-commerce sederhana, hingga desain sistem berbasis interaksi pengguna.

Tools yang Digunakan:
Figma (Desain UI/UX & prototyping)

Hasil & Pembelajaran:
Dari serangkaian proyek ini, tim mempelajari:
Cara merancang antarmuka yang memprioritaskan kenyamanan pengguna.
Pentingnya alur interaksi yang efisien dan jelas.
Bagaimana menerapkan prinsip UI/UX untuk berbagai tipe website dan kategori proyek.
Kolaborasi tim dalam menyusun sistem desain, dokumentasi, dan prototype.
